# Lunar explorer

## Problem

Artemis has landed a rover explorer onto the surface of the moon! The explorer needs to explore the suspiciously 2D
surface of the moon.

It does this by taking a string of commands telling it to either rotate left/right or move forward.

- `M` moves the explorer one space forward
- `L` rotates the explorer to the left / anti-clockwise
- `R` rotates the explorer to the right / clockwise

We know the size of the surface, the starting position of the explorer and the starting direction it is facing.

The surface is defined using X and Y coordinates. Starting at 0, 0 for the top left corner, with X representing the
horizontal size and Y the vertical size (suspiciously like computer screens).

---

## Acceptance Criteria

Your program must be able to:

* Take move commands and report the final position of the explorer

---

## Out of Scope

* Persisting position between runs (no saving or loading required)
